# Hue-Dial-Tap-Switch-Hubitat-Home-Assistant
Using a Philips Hue Tap Dial Switch  in Home Assistant via Hubitat C8 Pro as a 4 digit keypad for your whatever home automations
Create these helpers:
timer.alarm_rotary_cooldown 
input_number.alarm_mb1 (0 to 9)
input_number.alarm_mb2 (0 to 9)
input_number.alarm_mb3 (0 to 9)
input_text.alarm_locked_code
input_text.alarm_device_id_01
input_number.alarm_code_digit  (min  -1  to max 9)
timer.alarm_code_buffer_clear_timer
input_number.alarm_code_digit
No special settings with any of them
